## Misc

## Initialize a git respository
1. `git init`
1. `git init dir`

## Add to index
1. `git add <filename>`
1. `git add *`
1. `git add .`
1. `git add -A`

## Commit index
From index to create new commit object pointing to index changes and set branch to point to new commit 
1. `git commit -m "message"`
1. `git commit -a -m "message"`

## misc
